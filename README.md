# Trading Cuantitativo con R

## Objetivo

Optimizar un portafolio utilizando el Modelo de Markowitz y el Ratio de Sharpe.

## Activos Analizados

- Apple (AAPL)
- NVIDIA (NVDA)
- Amazon (AMZN)

## Herramientas

- R
- quantmod
- PerformanceAnalytics
- ggplot2
- dplyr

## Metodología

1. Obtención de datos desde Yahoo Finance.
2. Limpieza y validación.
3. Análisis exploratorio.
4. Correlación entre activos.
5. Optimización mediante Markowitz.
6. Simulación de 5000 portafolios.

## Resultado Final

| Activo | Peso |
|---------|---------|
| AAPL | 77.93% |
| NVDA | 21.54% |
| AMZN | 0.54% |

Sharpe Máximo: 0.1289
