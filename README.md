# Trading-Cuantitativo-con-R
Proyecto de optimización de portafolios utilizando el Modelo de Markowitz y el Ratio de Sharpe.  El objetivo es encontrar la mejor combinación de inversión entre:  Apple (AAPL) NVIDIA (NVDA) Amazon (AMZN)  utilizando datos históricos obtenidos desde Yahoo Finance y técnicas de análisis cuantitativo en R.
